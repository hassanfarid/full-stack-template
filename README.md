# Full Stack Template

The project contains template for bootstraping a full-stack project template based on,
- [Frontend (React)](./frontend/)
- [Backend API (Python - Flask)](./backend-api)
- [Datastore (Mongo, Postgres, Redis, MinIO)](.datastore/)
